Quick Start

Option 1: Simple API (Recommended)

<!-- Include the simple wrapper -->
<script src="https://cdn.jsdelivr.net/gh/pettarin/espeakng.js-cdn@latest/js/espeakng-simple.js"></script>

<script>
// Initialize TTS
const tts = new SimpleTTS({
  workerPath: 'https://cdn.jsdelivr.net/gh/pettarin/espeakng.js-cdn@latest/js/espeakng.worker.js'
});

// Wait for initialization
tts.onReady(function() {
  // Generate speech audio data
  tts.speak('Hello world!', function(audioData) {
    // audioData is a Float32Array - use your own audio player
    playAudio(audioData);
  });
  
  // With options
  tts.speak('Hello world!', {
    voice: 'en',
    rate: 200,
    pitch: 75
  }, function(audioData) {
    // Handle audio data
  });
});
</script>

Option 2: Original API

<!-- Include the original library -->
<script src="https://cdn.jsdelivr.net/gh/pettarin/espeakng.js-cdn@latest/js/espeakng.min.js"></script>

<script>
const tts = new eSpeakNG('path/to/espeakng.worker.js', function() {
  // Ready to use
  tts.synthesize('Hello world!', function(samples, events) {
    if (samples) {
      // Handle audio chunks
    }
  });
});
</script>

Required Files

You need these files (served from same origin or with proper CORS headers):

  • espeakng-simple.js or espeakng.min.js - Main library
  • espeakng.worker.js - Web Worker script
  • espeakng.worker.data - Voice data (must be in same directory as worker)

API Reference

SimpleTTS Methods

  • new SimpleTTS(options) - Create instance
  • onReady(callback) - Called when ready
  • getVoices(callback) - Get available voices
  • speak(text, [options], callback) - Generate speech
    • options.voice - Voice identifier
    • options.rate - Speech rate (80-450)
    • options.pitch - Voice pitch (0-100)
